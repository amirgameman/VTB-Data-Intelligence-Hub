# VTB Data Intelligence Hub

## Описание
Панель управления аналитикой данных ВТБ - интерактивное приложение для анализа клиентских данных банка ВТБ. Программа предоставляет инструменты для фильтрации, анализа и визуализации данных о клиентах.

## Технические требования
- Python 3.10 или выше
- Операционная система: Windows, Linux, macOS
- Минимальный объем оперативной памяти: 4 ГБ

## Установка и запуск

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. Запуск приложения
```bash
python main.py
```

## Настройки
Приложение поддерживает настройки через переменные окружения. Для настройки скопируйте файл `.env.example` в `.env` и отредактируйте параметры:

```bash
# Настройки темы оформления
APP_THEME=flatly

# Размеры окна приложения
APP_WINDOW_WIDTH=1400
APP_WINDOW_HEIGHT=900

# Настройки данных
DATA_SAMPLE_SIZE=100
```

## Функциональность

### Основные модули
- Финансовый обзор - сводная информация по балансам и доходам
- Демографический анализ - статистика по возрасту и регионам
- Анализ продуктов - распределение по банковским продуктам
- AI рекомендации - интеллектуальные выводы и предложения

### Дополнительные возможности
- Умные фильтры по возрасту, региону, продукту
- Текстовый поиск по клиентской базе
- Построение аналитических графиков
- Прогнозные показатели
- Экспорт данных в CSV формат
- Переключение между светлой и темной темой

## Используемые технологии
- ttkbootstrap - библиотека для создания современного GUI интерфейса
- pandas - работа с табличными данными и анализ
- matplotlib - построение графиков и визуализация
- numpy - математические вычисления и генерация данных

## Особенности реализации
- Автоматическая генерация тестовых данных (100+ записей)
- Адаптивный интерфейс с поддержкой различных разрешений
- Обработка ошибок и валидация данных
- Поддержка русского языка интерфейса
- Локальное хранение данных в CSV формате
