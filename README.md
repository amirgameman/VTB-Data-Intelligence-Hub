# VTB Data Intelligence Hub

## Описание

VTB Data Intelligence Hub - это аналитическое приложение для анализа клиентской базы банка ВТБ. Программа предоставляет инструменты для фильтрации, анализа и визуализации данных о клиентах, включая финансовые показатели, демографические характеристики и продуктовую активность.

## Технические требования

- Python 3.10 или выше
- Операционная система: Windows, Linux или macOS
- Минимальный объем оперативной памяти: 4 ГБ
- Рекомендуемый объем оперативной памяти: 8 ГБ

## Установка

### 1. Клонирование репозитория

```bash
git clone <URL репозитория>
cd vtb-intelligence-hub
```

### 2. Создание виртуального окружения (рекомендуется)

**Windows:**
```bash
python -m venv venv
venv\Scripts\activate
```

**Linux/macOS:**
```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

## Запуск приложения

### Основной способ запуска

```bash
python main.py
```

### Альтернативный способ запуска

```bash
python -m main
```

## Конфигурация

### Переменные окружения

Создайте файл `.env` на основе `.env.example`:

```bash
# Копируем пример конфигурации
cp .env.example .env
```

Пример содержимого `.env.example`:
```
# Настройки приложения
APP_THEME=flatly
APP_LANGUAGE=ru
APP_WINDOW_WIDTH=1400
APP_WINDOW_HEIGHT=900

# Настройки данных
DATA_SAMPLE_SIZE=100
DATA_RANDOM_SEED=42

# Настройки интерфейса
ENABLE_DARK_MODE=false
AUTO_REFRESH_INTERVAL=30000
```

### Основные настройки

- **APP_THEME** - тема оформления (flatly/darkly)
- **APP_LANGUAGE** - язык интерфейса
- **DATA_SAMPLE_SIZE** - количество генерируемых тестовых записей
- **ENABLE_DARK_MODE** - включение темной темы по умолчанию

## Учетные данные

Приложение использует синтетические данные, поэтому не требует аутентификации. Все данные генерируются автоматически при запуске.

## Зависимости

Основные зависимости проекта:

- **ttkbootstrap==0.5.2** - современная библиотека для создания GUI
- **pandas==1.5.3** - работа с табличными данными
- **numpy==1.24.3** - математические вычисления

Полный список зависимостей с версиями находится в файле `requirements.txt`.

## Функциональные возможности

### Основные модули

- **Умные фильтры** - поиск и фильтрация клиентов по различным параметрам
- **Финансовый обзор** - сводная информация по балансам, доходам и активам клиентов
- **Демографический анализ** - статистика по возрасту и географическому распределению
- **Анализ продуктов** - распределение клиентов по банковским продуктам и уровням риска
- **AI рекомендации** - интеллектуальные рекомендации для клиентской работы

### Дополнительные функции

- **AI инсайты** - автоматическая генерация аналитических выводов
- **Прогнозы** - прогнозные показатели по клиентским балансам
- **Таблица данных** - просмотр полной клиентской базы в табличном формате
- **Смена темы** - переключение между светлой и темной темой оформления

## Структура данных

Приложение работает с синтетическими данными, содержащими следующие поля:

- ID клиента
- Имя
- Возраст
- Регион
- Доход
- Баланс
- Активы
- Количество транзакций
- Банковский продукт
- Стаж лояльности
- Уровень риска
- Дата последней активности
